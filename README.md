# gosumetry-static-twitch
Remember to edit the values in `index.js`!


To get your twitch token, use this website: https://twitchapps.com/tmi .

If you want to use a twitch "Application", create one here: https://dev.twitch.tv/console/apps and select "chat:read chat:edit" as scopes on this website: https://twitchapps.com/tokengen .

Also you can use https://twitchtokengenerator.com, if you prefer the gui.
